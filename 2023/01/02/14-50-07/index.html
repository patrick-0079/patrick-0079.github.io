配置中心

配置中心是为解决不同服务运行环境需要不同的一个微服务单元
常用的配置中心有

  • Spring cloud config
  • Apollo
  • Nacos

Nacos 启动

单机模式

  1. 下载 Nacos
  2. 在 bin 目录下执行 startup 脚本:
    startup.cmd -m standalone 或 startup.sh -m standalone, 其中 standalone 为单机模式启动
  3. 使用 shutdown.sh 或 shutdown.cmd 脚本关闭服务
